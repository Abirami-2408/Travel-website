
Travel Website 🌍✈️

Project Description

A responsive travel website developed using HTML, CSS, and Bootstrap. The website allows users to explore destinations, view services, browse a gallery, and book trips.

Features

- Responsive navigation bar
- Beautiful home page
- Destination section
- Services section
- Gallery section
- About section
- Booking form
- Mobile-friendly design

Technologies Used

- HTML5
- CSS3
- Bootstrap 5
